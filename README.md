# AIで株価を予測しよう！
「AIで株価を分予測しよう！」は、プログラミング初心者から投資に関心のある実務者まで、AIを活用した株価予測の基礎から実践までを学び、データ分析とAIモデル構築のスキルを身につけることを目指す講座です。  
株価データという身近で興味深いテーマを題材に、AIの仕組みを「理解する」だけでなく、実際に手を動かしながら「使える」スキルへと昇華させることができます。  
本講座では、株価データの取得方法から基本的な解析手法、そしてニューラルネットワークやRNN、SVMなどのAIモデルの活用まで、段階的に学んでいきます。  
開発環境にはGoogle Colaboratoryを使用するため、面倒な環境構築やソフトウェアのインストールは一切不要。ブラウザとGoogleアカウントさえあれば、すぐにAIを使った株価解析を始められます。  
  
注意: 本講座が提供するのは、株価の分析方法の解説のみです。実際に株式投資をしたい方は自己責任でお願いします。  
  
講座の内容は以下の通りです。  
Section1. 株価解析の環境を整える  
→ 開発環境Google Colaboratoryの使い方、Pythonの学び方、株価の取得方法を解説します。  
Section2. 簡単な株価予測と結果の評価  
→ 簡単な方法で株価を予測し、結果を評価します。  
Section3. 様々なAIモデルを試す  
→ ニューラルネットワーク、RNN、SVMなどの様々なAIモデルを株価解析に使用します。さらに、生成AIの活用の可能性を探究します。
